# Quickstart Guide

## Prerequisites
- Node.js (for Tailwind build)
- Git

## Setup
1. Clone the repository: `git clone <repo-url>`
2. Install dependencies: `npm install`
3. Build Tailwind CSS: `npm run build-css`
4. Open `src/index.html` in a web browser

## Development
- Edit `src/data.js` to update portfolio content
- Modify `src/css/styles.css` for custom styles
- Run `npm run watch-css` for live Tailwind updates

## Deployment
- Commit changes to GitHub
- Enable GitHub Pages in repository settings
- Site will be available at `https://<username>.github.io/<repo>`

## Testing
- Run Lighthouse audit on the deployed site
- Test responsiveness on various devices
- Validate accessibility with WAVE or similar tools
